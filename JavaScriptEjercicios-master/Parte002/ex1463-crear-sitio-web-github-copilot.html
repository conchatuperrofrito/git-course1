¿Qué es GitHub Copilot?

GitHub Copilot es una herramienta de código autocompletado que te ayuda a escribir código más rápido. Copilot usa el contexto de tu código para sugerir líneas completas de código que puedes insertar en tu editor. Copilot está disponible en Visual Studio Code, PyCharm, y JetBrains IDEs.
